# STRIVEX - Neo-Glass Arctic Edition

![STRIVEX Banner](https://images.unsplash.com/photo-1639322537228-f710d846310a?auto=format&fit=crop&q=80&w=1200&h=400)

## 🚀 Overview
**STRIVEX** is an elite collective of ambitious innovators dedicated to pushing the boundaries of engineering, AI, and cybersecurity. Built with a focus on disciplined collaboration and technical excellence, STRIVEX serves as an accelerator for talent and a launchpad for high-stakes technological advancement.

This version features the **Neo-Glass Arctic** theme—a pristine, high-tech light aesthetic feauturing frosted glassmorphism and iridescent accents.

## ✨ Key Features
- **Modern Tech Stack**: Built with React, Vite, and Tailwind CSS for blistering performance.
- **Neo-Glass Arctic Design System**: A custom-crafted aesthetic featuring arctic white backgrounds, cyan-violet blurs, and premium frosted glass components.
- **Iridescent Animations**: Fluid transitions and holographic text effects powered by Framer Motion.
- **Professional Roadmap**: A clear, semantic journey from foundation building to large-scale startup deployment.
- **Elite Founder Spotlight**: A minimalist, high-impact founders page focusing on professional connection and leadership.

## 🎨 Visual Identity
### The Arctic Palette
- **Arctic White**: `#f8fafc` (Primary Background)
- **Neo Cyan**: `#06b6d4` (Primary Accents)
- **Pearl Lavender**: `#8b5cf6` (Secondary Accents)
- **Frosted Glass**: Translucent white panels with heavy blur (`backdrop-blur-3xl`) for a premium feel.

## 🛠️ Tech Stack
- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## � Getting Started

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
